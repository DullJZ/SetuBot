# SetuBot 色图机器人
使用telegram查看色图
## 使用公开机器人（不保证可用性）
1. https://t.me/tbSetubot 跳转到telegram打开机器人
2. 输入/start 开始使用
3. 输入/setu 在数据库中所有色图中随机发送一张色图
4. 输入/setu r18 在数据库中所有色图中随机发送一张18禁色图

## 自建机器人
1. 克隆代码
2. 创建token.txt文件，内容为你的机器人token，格式为：
```
123456:abcdefgh
```
3. 安装pyTelegramBotAPI库
4. 运行main.py
